# Mi Portfolio

Portfolio personal desarrollado con React, Three.js, Framer Motion y TailwindCSS.

## Built With

- React.js
- Three.js
- Framer Motion
- TailwindCSS

## Live Demo

- Visita: [https://111adyy.github.io](https://111adyy.github.io)

## Getting Started

### Prerequisites

Asegúrate de tener Node.js `16.0.0` o superior y npm `9.0.0` o superior instalados.

### Setup

1. Clona el repositorio:
```bash
git clone https://github.com/111adyy/111adyy.github.io.git
cd 111adyy.github.io
```

## Install

- After opening the files in Visual Studio Code, open the **VS Code** integrated terminal and run the following commands:

```
npm install
```

This will install all the packages and dependencies used in the project.

## Usage

- Run the following command to start a local server:

```
npm run dev
```

This will open up the project on a browser on `http://localhost:5173/`

## Build

- To create a production build:

```
npm run build
```

- To preview the site as it will appear once deployed:

```
npm run preview
```

## Deploy

Para desplegar en GitHub Pages:

```bash
npm run deploy
```

Esto construirá el proyecto y copiará los archivos necesarios a la raíz del repositorio. Luego haz commit y push a GitHub.
